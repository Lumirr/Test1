# 理解'*','*args','**','**kwargs'
  https://blog.csdn.net/callinglove/article/details/45483097

通过一个函数调用来理解’*’的作用
定义一个含三个位置参数的函数”fun”.
>>> def fun(a,b,c):
...     print a,b,c
... 
1
2
3
传三个位置参数调用此函数

>>> fun(1,2,3)
1 2 3       #输出
1
2
可以看到出入三个位置参数调用此函数，会打印出三个参数

现在我们定义一个含三个整数的数列，并使用’*’
>>> l = [1,2,3]
>>> fun(*l)
1 2 3       #输出
